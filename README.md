🎮 GAME HUB

«Универсальный сборник мини-игр в формате WebApp / PWA.»

GAME HUB — это веб-приложение, объединяющее различные мини-игры в едином интерфейсе. Проект ориентирован на запуск через браузер, GitHub Pages и мобильные устройства.

---

🌟 Возможности

- Запуск прямо в браузере
- Не требует установки
- Работает на ПК и смартфонах
- Простая HTML-архитектура
- Быстрая загрузка
- Поддержка GitHub Pages
- Возможность дальнейшего превращения в PWA
- Основа для будущего игрового портала MetaSymbiote

🎯 Цели проекта

Создать единый игровой центр, который сможет:

- объединять мини-игры;
- работать как PWA;
- запускаться на Android;
- использовать локальное хранение прогресса;
- поддерживать систему достижений;
- поддерживать пользовательские профили;
- интегрироваться в экосистему MetaSymbiote.
